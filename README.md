# 役員報酬 最適化シミュレーター

中小オーナー会社向けに、役員報酬、配当、法人留保、税金、社会保険料のバランスを概算比較する静的HTMLツールです。
最初に試算方針と所得税の年分を選び、その後に利益額・報酬額・税率等を調整します。

## 使い方

`index.html` をブラウザで開くだけで利用できます。GitHub Pagesでもそのまま公開できます。

## 検証

```bash
npm run test
npm run build
```

## 主な前提

- 所得税は総合課税の速算表で概算します。
- 試算方針は、バランス型、総手残り最大、個人キャッシュ最大、法人資金温存、配当活用の5パターンです。
- 基礎控除・給与所得控除は、令和7年分、令和8・9年分、令和10年分以後を選択して概算します。
- 配当は非上場会社オーナーの総合課税、配当控除ありを主な想定にしています。
- 法人税等は画面上の実効税率で概算します。
- 社会保険は協会けんぽ兵庫支部の令和8年度料率を初期値にしています。健康保険・介護保険は令和8年3月分以後、子ども・子育て支援金は令和8年4月分以後の料率を通年試算用に置いています。
- 月額0円から低額の役員報酬案では、社会保険に加入しない場合の国民健康保険料・国民年金保険料を反映していません。
- 令和10年分以後の基礎控除・給与所得控除の最低保障額は、今後のCPI連動見直し予定があるため、実際の適用時には最新法令を確認してください。
- 住民税の調整控除、均等割の非課税限度額、防衛特別法人税などは簡略化しています。
- 顧問先ごとの所在地、健康保険組合、資本金、欠損金、役員給与規制、退職金設計などは別途確認が必要です。
