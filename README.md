# 役員報酬検討ツール

中小オーナー会社向けに、役員報酬、配当、法人留保、税金、社会保険料のバランスを概算比較する静的HTMLツールです。
最初に試算方針と所得税の年分を選び、その後に利益額・報酬額・税率等を調整します。
おすすめ月額の横に、個人手取りと法人留保の計算過程と推薦理由を表示します。
画面上部の「使い方」ボタンから、面談中に確認しやすい簡易マニュアルを開けます。
画面上部の「印刷」ボタンから、入力内容と試算結果を印刷できます。
入力欄は、方針、会社・役員、持株・その他所得、配当・評価、詳細設定に分類して確認できます。
「内部留保から逆算」方針では、方針選択の直下に表示される目標法人留保を先に入力し、その金額に近い候補を上位に表示します。数値・金額はスライダーを使わず、各入力欄へ直接入力します。

## 使い方

`index.html` をブラウザで開くだけで利用できます。GitHub Pagesでもそのまま公開できます。

## 検証

```bash
npm run test
npm run build
```

## 主な前提

- 所得税は総合課税の速算表で概算します。
- 試算方針は、バランス型、内部留保から逆算、総手残り最大、個人キャッシュ最大、法人資金温存、配当活用の6パターンです。
- 試算方針の配下項目を手動変更した場合、方針表示はカスタムに切り替わります。
- 役員構成は、単独役員と夫婦役員を選択できます。夫婦役員モードは合計月額報酬を固定した配分探索と、役員A/Bそれぞれの月額探索に対応します。
- 夫婦役員モードの数式最適解は、税務上の妥当性を保証するものではありません。実質的な業務内容と報酬額の整合性は別途確認が必要です。
- 基礎控除・給与所得控除は、令和7年分、令和8・9年分、令和10年分以後を選択して概算します。
- 配当は非上場会社オーナーの総合課税、配当控除ありを主な想定にしています。
- 法人税等は画面上の実効税率で概算します。
- 社会保険は協会けんぽ令和8年度の都道府県別料率を選択できます。初期値は兵庫支部です。健康保険・介護保険は令和8年3月分以後、子ども・子育て支援金は令和8年4月分以後の料率を通年試算用に置いています。
- 報酬ゼロから低額＋配当の候補は、社会保険加入状況、住民税の細部、分配可能額を個別に確認してから採用してください。
- 月額0円から低額の役員報酬案では、社会保険に加入しない場合の国民健康保険料・国民年金保険料を反映していません。
- 令和10年分以後の基礎控除・給与所得控除の最低保障額は、今後のCPI連動見直し予定があるため、実際の適用時には最新法令を確認してください。
- 住民税の調整控除、均等割の非課税限度額、防衛特別法人税などは簡略化しています。
- 顧問先ごとの所在地、健康保険組合、資本金、欠損金、役員給与規制、退職金設計などは別途確認が必要です。
